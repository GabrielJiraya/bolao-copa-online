# Bolao da Copa Online

Site simples para bolao por dia:

- participante envia nome e placar dos jogos do dia;
- palpites ficam escondidos;
- admin cadastra jogos, lanca resultados e libera ranking;
- dados ficam em `data/db.json`.

## Rodar

```bash
npm start
```

Admin PIN padrao: `1606`.

Para trocar:

```bash
ADMIN_PIN=1234 npm start
```
